# 📊 Data Analysis – Week 4

This repository contains analysis performed on the Titanic and Housing datasets.

## 🔍 Topics Covered
- Descriptive Statistics
- Probability Distributions
- Hypothesis Testing (t-test, ANOVA, correlation)
- Correlation vs Causation

## 📁 Datasets
- Titanic Dataset
- Housing Dataset

## 🛠️ Tools Used
- Python (Pandas, NumPy)
- Seaborn, Matplotlib
- SciPy

## 📈 Key Insights

### Titanic
- Gender and passenger class significantly affected survival
- Higher fare increased survival probability

### Housing
- Area and location strongly influence house prices
- Features like air conditioning and furnishing increase value
